Basic Tags

The Paragraph <p> Tag

This is a paragraph. Paragraphs are blocks of text and/or images and form fields. Paragraphs are block elements and will automatically close if another block-level element is parsed before the closing of the </p> element. <p> is the opening tag of the p element and </p> is the closing tag of the p element. They relate to one another because thay set the boundaries of the paragraph element and anything within the <p> and the </p> tags will show up as text, images, forms, etc.

The <strong> and <b> Tags

The <strong> tag is used to mark text with strong importance. The content inside this tag is typically displayed in bold. The <b> tag is used to show words in bold without marking them as important.

The <a> Tag

The HTML <a> element or anchor element, with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each <a> should indicate the link's destination.

Want to learn more about the anchor tag? Try HTML <a> Tag!

The <img> Tag

The <img> Tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages. The <img> tag creates a holding space for the referenced image. This tag also requires two attributes which are the src and alt attributes. The src attribute specifies the path to the image while the alt attribute specifies an alternative text the image, if the image for some reason cannot be displayed.
