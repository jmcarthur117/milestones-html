SNIPPET

Want to search the web? Try Google!

The above code looks like this: <p> Want to search the web? Try <a href="https://google.com" id="the-best-link" class="banana" target="_blank">Google</a>! </p>

There are are two tags in the above section. The first tag is the paragraph tag/element which has the opening tag of <p> and closing tag of </p>. These tags are used to form blocks. Paragraphs are blocks of text and/or images and form fields. Paragraphs are block elements and will automatically close if another block-level element is parsed before the closing of the </p> element. <p> is the opening tag of the p element and </p> is the closing tag of the p element. They relate to one another because thay set the boundaries of the paragraph element and anything within the <p> and the </p> tags will show up as text, images, forms, etc.

The second tag in the code above is the <a> tag. The HTML <a> element or anchor element, with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each <a> should indicate the link's destination.

There are three attributes in the <a> above. The three attributes are id, class, and target.

The id attribute specifies a unique id for an HTML element (the value must be unique within the HTML document). The id attribute is most used to point to a style in a style sheet, and by JavaScript (via the HTML DOM) to manipulate the element with the specific id.
The id attribute above has a value of "the best link", which gives it the id of "the best link"

The class attribute specifies one or more classnames for an element. The class attribute is mostly used to point to a class in a style sheet. However, it can also be used by a JavaScript (via the HTML DOM) to make changes to HTML elements with a specified class.
The class attribute above has a value of "banana", which means it will go to a css or java script file of banana.

The target attribute specifies where to open the linked document. There are four target attributes that can be used. "_blank" Opens the linked document in a new window or tab. "_self" Opens the linked document in the same frame as it was clicked (this is default). "_parent" Opens the linked document in the parent frame. "_top" Opens the linked document in the full body of the window.
The value in the target attribute above is "_blank", which means it will open in a new window or tab.
