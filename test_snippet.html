PARAGRAPH ELEMENT

In HTML their is the <p> element which represents a paragraph. Paragraphs are blocks of text and/or images and form fields. Paragraphs are block elements and will automatically close if another block-level element is parsed before the closing of the </p> element. <p> is the opening tag of the p element and </p> is the closing tag of the p element. They relate to one another because thay set the boundaries of the paragraph element and anything within the <p> and the </p> tags will show up as text, images, forms, etc.

THE ANCHOR ELEMENT

The HTML <a> element or anchor element, with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each <a> should indicate the link's destination.

HTML ATTRIBUTES

All HTML elements can have attributes and attributes provide additional information about elements. Attributes are always specified in the start tag and usually come in name/value pairs like: name="value"

THE href ATTRIBUTE

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

href EXAMPLE

Visit W3Schools for more info
Example Code: <a href="https://www.w3schools.com">Visit W3Schools for more info</a>

THE src Attribute

The img tag is used to embed an image in an HTML page. The src attribute specifies the path to the image that you wish to display. You can also use width and height attributes within an img tag. These attributes specifie the width and height of an image in pixels.

Example Code: <img src="james.jpg" width="500" height="600" alt="A Picture Of James">

THE alt ATTRIBUTE

The alt attribute is required for the <img> tag because it specifies an alternative text for an image. If an image for some reason cannot be displayed on your site the alt attribute for the <img> will show text instead. Errors with an image can occur with slow connection, an error in the src attribute, or if the user uses a screen reader. The alt attribute is shown in the example code above.
